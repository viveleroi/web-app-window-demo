For this demonstration we would like for you to develop a functional application window on a web page. This window should contain basic look and functional behaviors of a typical application window.

Goals:

  1. Use only JavaScript, HTML, and CSS.
  2. Everything must be written by you and you alone (no libraries).
  3. Window should look like an application window
  4. Window should be able to hide/show
  5. Window should be able to capable of being dragged over the screen
  6. You are free to add any other features outside of the basic requirements (feel free to be creative)
  7. Make sure everything is packaged up so we can view the working product on our end (this includes all of the files)
